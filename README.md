# Uppskriftavefur

## Hugmyndalýsing
Þetta verkefni er einfaldur uppskriftavefur þar sem notendur geta skoðað uppskriftir. Markmiðið með þessum vef er að æfa uppsetningu á bakenda með Node.js og Express, nota EJS sem view engine til að render-a HTML viðmót, og nýta SASS fyrir stílhreint og skalanlegt útlit.

## Gögn sem við ætlum að geyma
Í framtíðinni (þegar gagnagrunnur bætist við) er hugmyndin að vista eftirfarandi gögn um hverja uppskrift:
- **Nafn (Title):** Heiti á uppskriftinni (t.d. "Kanilsnúðar ömmu").
- **Hráefni (Ingredients):** Listi af hráefnum sem þarf.
- **Tími (Time/Duration):** Hversu langan tíma tekur að undirbúa og elda.
- **Mynd (Image):** Slóð á mynd eða skráin sjálf sem sýnir útkomuna.

*Þetta verkefni var unnið sem liður í vikulegu markmiði í Vefþróun (Vefþjónn, EJS, SCSS).*
